# 🤖 RAG-Based AI PDF Question Answering Assistant

An AI-powered web application that allows users to upload PDF documents 
and ask questions in natural language, receiving accurate answers 
generated by Google Gemini AI.

## ✨ Features
- 📄 Upload any PDF document
- ❓ Ask questions in plain English
- 🤖 Get AI-powered answers instantly
- 🕐 View chat history
- 🎨 Clean, modern UI

## 🛠 Tech Stack
- Python
- Google Gemini AI (gemini-2.0-flash)
- LangChain
- Streamlit
- PyPDF2

## 🚀 How to Run
1. Clone this repository
2. Install dependencies: `pip install -r requirements.txt`
3. Create `.env` file and add: `GOOGLE_API_KEY=your_key_here`
4. Run: `streamlit run app.py`
